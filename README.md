# prototype

Based on: https://github.com/patlux/bun-v1.2-new-bundler-boilerplate

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

## prepare local DB

Python is used to create a SQLite3 file called `dataset.sqlite3`
### setup

python3 scripts/create_db.py
